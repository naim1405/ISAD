# Data Entities – Nextstepbd

The current system handles several key data entities:

## 1. Client
- Client ID
- Name
- Contact information
- Requirements history

## 2. Project
- Project ID
- Project name
- Description
- Status
- Timeline

## 3. Task
- Task ID
- Task description
- Assigned developer
- Deadline
- Status

## 4. Developer
- Developer ID
- Name
- Skills
- Assigned tasks

## 5. Report
- Report ID
- Project summary
- Output files
- Delivery status
