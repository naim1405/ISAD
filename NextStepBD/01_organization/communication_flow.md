# Communication Flow – Nextstepbd

The communication system is primarily based on multiple tools:

## Internal Communication
- WhatsApp (main tool for daily communication)
- Slack (used for international or structured discussions)

## External Communication
- WhatsApp (primary client communication channel)
- Email (used for formal communication and final reporting)

## Flow Structure
Client → Project Manager → Developers → Project Manager → Client

## Observations
- Communication is fast but unstructured
- Important information is often stored in chat messages
- Lack of centralized communication system
