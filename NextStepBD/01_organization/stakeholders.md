# Stakeholders – Nextstepbd

The key stakeholders involved in the system are:

## 1. Clients
- Provide project requirements
- Approve deliverables
- Give feedback during development

## 2. Project Managers
- Act as intermediaries between clients and developers
- Assign tasks to development teams
- Monitor project progress

## 3. Developers
- Responsible for software development
- Use GitHub for version control
- Implement client requirements

## 4. Internal Team Leads
- Supervise development teams
- Ensure quality and deadlines are met

## 5. Administrative Staff
- Handle documentation and reporting
- Manage internal coordination and communication
